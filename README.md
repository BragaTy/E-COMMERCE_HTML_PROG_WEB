# E-COMMERCE_HTML_PROG_WEB

## Projeto: E-Commerce Game Station

Trabalho prático de desenvolvimento web. 
O projeto consiste na criação de um site para uma loja virtual de games, aplicando os conceitos ensinados em sala de aula.

## Alunos
- Matheus Braga

## Arquivos do Projeto
O site possui 5 páginas funcionais:
- `index.html` (Página Inicial)
- `catalogo.html` (Catálogo de Produtos)
- `Carrinho.html` (Carrinho de Compras)
- `faleconosco.html` (Página de Contato/Formulário)
- `noticia_ps5.html` (Página de Notícia)

## Instruções

1. Faça o download ou clone este repositório.
2. Certifique-se de que todas as imagens (`.png` e `.jpg`) estão na mesma pasta que os arquivos HTML.
3. Dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador de preferência (Chrome, Firefox, Edge, etc).
4. Navegue pelo site através do menu no cabeçalho.
